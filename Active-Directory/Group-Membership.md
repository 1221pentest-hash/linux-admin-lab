# Active Directory Group Membership

## Objective

Implement Role-Based Access Control (RBAC) by assigning user accounts to departmental security groups.

## Security Groups

- GG_HR_Users
- GG_Finance_Users
- GG_Sales_Users
- GG_IT_Users
- GG_Operations_Users

## Benefits

- Simplifies permission management
- Improves scalability
- Follows the Principle of Least Privilege
- Supports centralized administration

## Result

All employee accounts were assigned to the appropriate departmental Global Security Group.

Permissions will be assigned to groups rather than directly to users.
